# Investment Research Desk Hero V5 Script

## Goal
Keep the improved brand opening and cleaner professional market field from v4, but replace phase 3 with a **multi-agent team scene**.

This scene should feel like the introduction of a high-end investment-bank research team, but translated into a low-density motion-graphics language for a GitHub README hero.

## Core memory points
After one viewing, a new user should retain:
- this is a finance research product
- it works through a specialist team, not a single generic chat box
- the team is built around:
  - macro / fundamental
  - news impact
  - sentiment
  - technical
- the product behaves like a structured research workflow
- it is research support, not execution

## Phase structure

### Phase 1 — Institutional brand reveal
- Keep the current refined v4 opening style.
- Keep the bottom positioning line.
- Calm, bank-grade, premium.

### Phase 2 — Professional market structure field
- Keep the current finance/professional market field idea.
- K-line, price path, macro pressure, sparse ticker rail.
- Avoid extra clutter.

### Phase 3 — Multi-agent research team introduction
Do not show a dashboard.
Do not show a full workflow board.
Do not show paragraphs.

Instead, create a composed “team presentation” memory image:
- a central label such as `Analyst Team` or `Research Team`
- four low-density specialist cards / plaques / nameplates around it:
  - `Fundamental / Macro`
  - `News Impact`
  - `Sentiment`
  - `Technical`
- each one should have only a tiny supporting phrase, not detailed copy
- use layout and motion to feel like an elite research desk / bank team roll-call

## Motion direction for phase 3
- Phase 2 chart contracts and dims into the background.
- Four team plates emerge sequentially but smoothly, not like popping UI.
- Use subtle line/ring connectors toward the center so it feels coordinated.
- Final hold should show:
  - the team concept
  - the four specialist functions
  - a clear footer that this is research context only

## Constraints
- Low information density
- Large, readable labels only
- No small paragraphs in phase 3
- No dashboard blocks in phase 3
- GIF-safe transitions only
