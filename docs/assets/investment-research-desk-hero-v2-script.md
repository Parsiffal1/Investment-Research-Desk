# Investment Research Desk Hero V2 Script

## Goal
Create a second README hero animation that is more explicitly finance-coded than the current abstract premium version, while still feeling high-end and product-like.

This version should make it instantly obvious that the product helps with **market research** across equities and crypto.

## Creative direction
- Keep the dark, premium, institutional aesthetic.
- Make **financial market structure** much more legible.
- Use visible **K-line / candlestick**, **price path**, **ticker**, **funding / OI / macro pulse** motifs.
- Preserve the product’s positioning as a **research assistant**, not a trading bot.

## Storyboard

### Phase 1 — Brand ignition
- Dark stage with subtle market-grid texture.
- Thin price-line pulses and small ticker fragments appear in the background.
- `Investment Research Desk` resolves from market noise.
- Subtitle frames it as a local research workflow for equities and crypto.

### Phase 2 — Market field formation
- A large chart field assembles.
- Visible financial motifs appear:
  - candlestick bands
  - price curve
  - volume bars
  - macro pulse ribbon
  - ticker labels like `SPY`, `QQQ`, `ETH-USDT-SWAP`, `funding`, `OI`, `CPI`
- The feeling should be: the system is watching cross-asset market structure.

### Phase 3 — Research assist layer
- Overlay focused research-assistant semantics on top of the market field:
  - `macro context`
  - `news impact`
  - `sentiment`
  - `technical state`
  - `bull / bear debate`
- These should feel like interpretive lenses over the market, not just random floating labels.

### Phase 4 — Context convergence
- The market field compresses into a structured final research board.
- Final board should emphasize:
  - directional view
  - risk / confidence
  - research context only
  - traces / metrics / artifacts
- End on a clear message that this is a **financial research support product**, not an execution engine.

## Visual rules
- Finance motifs should be readable at README scale.
- Avoid noisy micro-text and over-dense panels.
- In GIF export mode:
  - no animated film grain
  - no backdrop blur
  - minimize shimmer-prone gradients
  - use stable large-shape motion

## Export rules
- Use `scripts/render_true30_gif.cjs`
- Keep this version separate from the current hero asset
- New asset names:
  - `docs/assets/investment-research-desk-hero-v2.html`
  - `docs/assets/investment-research-desk-hero-v2.gif`
